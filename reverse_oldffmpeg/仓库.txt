简易的命令行入门教程:
Git 全局设置:
git config --global user.name "姓名"
git config --global user.email "邮箱"

创建 git 仓库:
mkdir video-reverse-player
cd video-reverse-player
git init
touch README.md
git add README.md
git commit -m "first commit"
git remote add origin https://gitee.com/dsxiaolong/video-reverse-player.git
git push -u origin master

已有仓库?
cd existing_git_repo
git remote add origin https://gitee.com/dsxiaolong/video-reverse-player.git
git push -u origin master
